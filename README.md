# 🌦️ Weather Analytics Dashboard

A full-stack weather data system that collects, stores, and visualizes real-time and historical weather data for multiple cities.

## 🚀 Features
- Real-time weather data ingestion
- MySQL database storage
- REST API (Flask)
- Multi-city comparison
- Forecast & historical trends
- Scheduled updates (APScheduler)

## 🧱 Tech Stack
- Backend: Python (Flask)
- Database: MySQL
- Data Processing: pandas
- Scheduler: APScheduler
- Caching: Redis (optional)
- Frontend: HTML / JavaScript / React (optional)
